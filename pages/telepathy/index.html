--- title: 心灵感应 ---

概述

本页面记载了我在使用软件时，尽管没有阅读过相关说明文档，却一次命中不明显的用法达成需求的历史。

例如，没有触摸设备经验的用户首次使用iPhone迅速上手是寻常的事，但没有命令行经验的用户首次接触外壳就能自如地导航是骇人的新闻。本页的目的就在于记录我体验到的后一种情况。

这种现象的发生表明软件的设计存在良好的可预测性，却缺乏更直接的引导，以至于成功利用可预测性都是一种成就。

列表

  • 在Windows“运行”对话框中输入：
    • devmgmt.msc = 设备管理器。
    • diskmgmt.msc = 磁盘管理。
    • Control Panel = 控制面板。这是因为控制面板的映像名为control.exe，启动时忽略了参数。
  • 在Microsoft Word度量值输入框中可以手动输入与预设值不同的单位。这一点在其他语言版本中显而易见，但在简体中文版中，单位缩写被本地化为汉字（例如，不能输入“cm”，必须输入“厘米”），极为不便。我从未使用过简体中文以外版本的Office套件。
  • 在osu!谱面列表可以使用高级搜索语法，例如bpm:114以搜索114 BPM的歌曲，尽管界面上没有任何暗示。（2025-06-09）
  • GitHub discussions有以下未公开记载的REST API端点。发现于编写评论系统的过程中。（2025-06-26）
